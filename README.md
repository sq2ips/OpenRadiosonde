# OpenRadiosonde

This project has a goal of making a open source amateur radiosonde project for high altitude balloon flights with telemetry and images downlink.

# Main features:
  - Dual band 100mW transmitter 2m and 70cm
  - Horus Binary V3 4FSK and AFSK APRS telemetry encoder
  - SSTV encoder
  - camera and other sensors connector
  - Battery voltage ADC
  - GPS positioning

# Hardware
